## Install

```
$ npm install iquotes-cli
```

## Usage

```js
$ iquotes
// "If you really want to do something, you'll find a way. If you don't, you'll find an excuse." - Jim Rohn (Life)
$ iquotes --love
// "The first duty of love is to listen." - Paul Tillich (Love)
```
